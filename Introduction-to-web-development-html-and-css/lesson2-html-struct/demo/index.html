Módulo 1: Fundamentos

El esqueleto de la Web

Aprende a dominar las etiquetas y la estructura esencial que toda página web necesita para existir.

Anatomía de una Etiqueta

Cada elemento en HTML es una instrucción clara para el navegador.

  • Apertura: Inicia el elemento <tag>
  • Contenido: Lo que el usuario lee.
  • Cierre: Finaliza con una barra </tag>
<!-- Ejemplo -->
<h1>Mi Título</h1>

Estructura Estándar

Este es el código que usarás en absolutamente todos tus proyectos.

<!DOCTYPE html>
<html>
  <head>
    <title>Mi Página</title>
  </head>
  <body>
    <h1>Hola Mundo</h1>
    <p>Primer párrafo.</p>
  </body>
</html>
