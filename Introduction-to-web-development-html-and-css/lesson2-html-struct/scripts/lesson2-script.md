### Lección 2: Estructura HTML básica

> Conoce las etiquetas esenciales y crea tu primera página web desde cero.

- **Video**: [Ver la lección](https://www.youtube.com/watch?v=enlace-leccion-2)
- **Conclusión clave**: Toda página HTML tiene `<html>`, `<head>`, `<body>` como base.
- **Práctica**: [Abrir en CodePen](https://codepen.io/pen/) — sin registro

#### Objetivo
Comprender la estructura base de un documento HTML y usar las etiquetas más comunes.

#### Estructura base

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Mi primera página</title>
  </head>
  <body>
    <h1>Hola mundo</h1>
    <p>Esta es mi primera página web.</p>
  </body>
</html>
```

#### Etiquetas principales

| Etiqueta | Para qué sirve |
|----------|----------------|
| `<h1>` … `<h3>` | Títulos de diferente tamaño |
| `<p>` | Párrafos de texto |
| `<strong>` | Texto en negrita |
| `<em>` | Texto en cursiva |
| `<br>` | Salto de línea |

#### Consejos
- Siempre cierra tus etiquetas — `<p>texto</p>`, no solo `<p>texto`.
- El `<head>` es para configuración, el `<body>` es para lo que se ve.
- Prueba cambiar los textos y observa cómo cambia la vista previa al instante.
