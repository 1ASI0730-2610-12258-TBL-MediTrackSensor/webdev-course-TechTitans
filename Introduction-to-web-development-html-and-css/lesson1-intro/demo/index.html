Módulo 1: Introducción

¿Qué es un sitio web?

Descubre qué sucede detrás de escena cuando navegas por internet y cómo el HTML y el CSS dan vida a tus sitios favoritos.

🧱

HTML: El Esqueleto

Es el lenguaje que define el **contenido** y la **estructura**. Sin él, no habría títulos, imágenes o párrafos.

<h1>Hola Mundo</h1>
🎨

CSS: El Estilo

Se encarga del **diseño** y la **apariencia**. Colores, fuentes, espaciados y animaciones que hacen que una web sea impactante.

color: blue; font-size: 20px;

"HTML es la casa, CSS es la decoración."
