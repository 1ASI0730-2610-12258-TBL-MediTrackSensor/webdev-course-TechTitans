# 🚀 Lección 1: ¿Qué es un sitio web?

¡Bienvenido a la primera lección del curso! Aquí aprenderás los fundamentos básicos de la web.

## 📁 Contenido de esta carpeta

- **`/scripts`**: Contiene el guión detallado para la lección (`lesson1-script.md`).
- **`/demo`**: Una demostración visual interactiva de los conceptos de HTML y CSS.

## 🎯 Objetivos de la lección
1. Entender qué es un sitio web desde la perspectiva del navegador.
2. Diferenciar entre la estructura (**HTML**) y el estilo (**CSS**).
3. Comprender la analogía de la "casa" para facilitar el aprendizaje.

## 💻 Visualización del Demo
Para ver el diseño moderno propuesto para esta lección, abre el archivo `demo/index.html` en tu navegador.
