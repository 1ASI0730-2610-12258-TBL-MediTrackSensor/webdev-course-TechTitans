---

## Introducción 
> "¡Hola a todos! Bienvenidos a este curso de desarrollo web.  
> Hoy vamos a descubrir un secreto:  
> **¿qué hay realmente detrás de un sitio web cuando lo abres en tu navegador?**"

---

## ¿Qué es un sitio web? 
"Un sitio web es un conjunto de páginas que puedes explorar desde cualquier navegador — *Chrome, Firefox o Safari*.  
Piensen en los sitios que usan a diario:"

*   **YouTube** → para ver videos.
*   **Wikipedia** → para buscar información.
*   **Instagram** → para ver fotos y publicaciones.
*   **Amazon** → para comprar productos.

"Todos esos sitios están creados con las mismas dos herramientas: **HTML y CSS**."

---

## HTML vs CSS 
"¡Imaginen que estamos construyendo una casa!"

*   **HTML es la estructura** — las paredes, el techo y las puertas.
*   **CSS es la decoración** — los colores, los muebles y el estilo.

> **Nota visual:** Mostrar una comparación de una casa en obra negra vs. una casa terminada y decorada.

"Sin HTML no hay contenido. Sin CSS el contenido existe pero se ve aburrido, como un texto plano."

"Por ejemplo, el título que ves al entrar — eso es **HTML**. Que ese título sea azul, grande y con una fuente atractiva — eso es **CSS**."

---

## ¿Cómo funciona realmente?
"Cuando abres una página web, tu navegador descarga dos tipos de archivos: el **HTML** y el **CSS**.  
El navegador los lee y los transforma en todo lo visual que ves."

"Nosotros vamos a aprender a escribir esos archivos desde cero, directo en el navegador y sin instalar nada."
